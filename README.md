# NASty Savings

A small single-page calculator for figuring out how much you're saving by self-hosting instead of paying for cloud services. Toggle the services you run, plug in your hardware and running costs, and it shows your monthly savings and break-even timeline.

Built for personal use, shared in case it's useful to anyone else doing the same math.
